# 📝 Task Manager

A clean, responsive, and lightweight Task Management web application built using native **Vanilla JavaScript**, HTML5, and CSS3. This project demonstrates modern frontend basics, state management via LocalStorage, and interactive DOM manipulation without the use of heavy external frameworks.

## 🚀 Live Demo
Link : https://rehanabanoo.github.io/Task_Manager/?

---

## ✨ Features

- **Persistent Storage:** Uses `localStorage` to save your tasks so your data remains intact even after refreshing the page.
- **Dynamic Filters:** Easily toggle between viewing **All**, **Pending**, and **Completed** tasks to stay organized.
- **Task Analytics:** Live counter tracking the total number of remaining items left to complete.
- **Bulk Cleanup:** A "Clear Completed" utility to quickly flush out finished tasks from your layout.
- **Security First:** Features built-in HTML character escaping to securely handle task inputs and prevent basic Cross-Site Scripting (XSS) injection attempts.
- **Responsive UI:** Fully optimized layout styled beautifully for both mobile screens and desktop monitors.

---

## 🛠️ Tech Stack

- **HTML5:** Semantic markup structure.
- **CSS3:** Custom UI variables, clean transitions, responsive flexbox modules, and micro-interaction animations.
- **JavaScript (ES6+):** Complete client-side functionality utilizing event delegation, state arrays, and native Web APIs.
